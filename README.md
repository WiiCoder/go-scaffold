## 目录结构
```shell
.
├── api             # API接口
├── config          # 配置文件
├── docs            # swagger 接口文档
├── global          # 全局参数
├── initialization  # 初始化
├── resources       # 资源文档
├── router          # 路由
├── service         # 服务
└── utils           # 通用工具类
```
